# Screenshots

This folder is populated by the engine screenshot harness:

```sh
./tools/bootstrap.sh          # downloads Godot, verifies, captures PNGs here
```

It could not be pre-populated in the build environment because the Godot
binary hosts are blocked by its egress policy (see ARCHITECTURE.md →
"Verification strategy"). For a preview of the intended look, see the
HD-2D reference shots from the playable web prototype of the same game in
[`web-prototype/docs/screenshots/`](../../web-prototype/docs/screenshots/).
